#### Improve comment format (#9672 by @fisker)

When multiple comments on same line, the result became unstable.

<!-- prettier-ignore -->
```jsx
// Input
a;
/*1*//*2*/
/*3*/
b;

// Prettier stable
a; /*2*/
/*1*/ /*3*/
b;

// Prettier stable (second format)
a; /*2*/ /*3*/
/*1*/ b;

// Prettier main
a;
/*1*/ /*2*/
/*3*/
b;
```
